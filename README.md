<h3><i>Invitation</i></h3>

<p>
<em>
Dare to taste my fare, dear diner!<br>
Come tomorrow it tastes finer<br>
and day after even good!<br>
If you still want more - I'll make it,<br>
from past inspiration take it,<br>
turning food for thought to food.
</em>
</p>
<p><sub>&mdash; Nietzsche</sub></p>

A fully managed, cross-platform image manipulation library with encoder/decoder for <code>.ani</code>, <code>.bmp</code>, <code>.jpg</code>, <code>.png</code>, <code>.gif</code>, <code>.ico</code>, <code>.cur</code>, <code>.jp2</code>, <code>.webp</code>, <code>.pnm</code>, <code>.qoi</code>, <code>.tga</code>, and <code>.tiff</code>, animations and lossy/lossless features of formats are supported.<br><br>
<a href="https://github.com/lofcz/morpheus"><b>Morpheus</b></a><br>
A morphological engine for declension of Czech names from nominative to vocative, a hybrid of a trained model, statistically extracted rules, and a precomputed index. Benchmarked to correctly handle over 98% of inputs from a 1,000 "write your name here" dataset. Handles titles, foreign and indeclinable names, compound names, nicknames, and many other edge-cases.<br><br>
<a href="https://github.com/lofcz/minfold"><b>Minfold</b></a><br>
An opinionated scaffolder & migrations generator for EF Core with plugins implemented for <a href="https://plugins.jetbrains.com/plugin/23315-minfold">JetBrains</a> and <a href="https://marketplace.visualstudio.com/items?itemName=lofcz.minfold">VisualStudio</a>. Minfold supports type-augmented mapping natively, eg. <code>enum</code> (C#) to <code>int</code> (DB).<br><br>
<a href="https://github.com/lofcz/tereo"><b>TeReo</b></a><br>
A modern replacement for <code>Resource</code>-based localization. Generates strongly typed C# & TS code, ideal for fullstack development. Includes native Blazor application in Avalonia shell for keys management. Highlights: one-click AI translate, infinite undo/redo, <a href="https://github.com/lofcz/fastdex">instant search</a>, key groups, lazy-loaded client modules.<br><br>
<a href="https://github.com/WattleScript/wattlescript"><b>WattleScript</b></a><br>
A scripting engine with dual-language environment, providing support for Lua 5.2 code as well as its own language, Wattle, a fusion of C#, JS, and Lua. Includes a safe sandbox, <code>async</code> interop. Scripts can be precompiled to bytecode and executed on a fully managed, stack-based VM. Co-authored with <a href="https://github.com/CallumDev">CallumDev</a>, who implemented the hardest parts.<br><br>
<a href="https://github.com/lofcz/Json5Core"><b>Json5Core</b></a><br>
